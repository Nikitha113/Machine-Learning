# Machine-Learning - Logistic Regression
This project uses Logistic Regression to predict whether a student will pass or fail based on study hours. It includes data creation, label encoding, train-test split, model training, performance evaluation, and visualization using Matplotlib to show how study hours affect results.

This project predicts whether a student will pass or fail based on study hours
using Logistic Regression.
## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab
